# mel

TIL(Today I Learning) + Memo

Can look back my daily notes.

## Uses

1. Specification of til
```
$ mel init

[ o ] Init Complete
```

2. show init list
```
$ mel list

[ ! ] Can Use Commands!!

[ * ] /mel/README.md
[ * ] /modules/a.md
[ * ] /modules/README.md
```

3. show tils  
`mel [list_line]`
```
$ mel /mel/README.md
```
