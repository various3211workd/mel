## Usage

#### Specification of til
```
PS>mel init

[ o ] Init Complete
```

#### show init list
```
PS>mel list

[ ! ] Can Use Commands!!

[ * ] /mel/README.md
[ * ] /modules/a.md
[ * ] /modules/README.md
```

#### show tils  
```
PS>mel [list_line] # or mel -n [list_number]

$ mel /mel/README.md
```
